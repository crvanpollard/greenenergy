# Mapbox Table Join (local data)
Mapbox sample with locally stored CSV table join to style map.

## JS library dependencies

- [mapbox-gl.js (v1.11.0)](https://docs.mapbox.com/mapbox-gl-js/api/)
- [bootstrap (v3.0.0)](https://getbootstrap.com/docs/versions/)
- [jquery (v1.10.2)](https://api.jquery.com/)

## Geospatial Data dependencies

- [DVRPC Vector Tiles](https://tiles.dvrpc.org/)
